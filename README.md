INSTRUÇÕES DE USO

INSTALAR A EXTENSÇAO NO CHROME

1 - Copiar a pasta extensions_feegow_chrome para o C:/

2 - Abrir o Chrome e acessar chrome://extensions/

3 - No canto superirior direito ativar o <Modo Desenvolvedor>

4 - Clicar em "Carregar sem compactação"

5 - Escolher a pasta C:\extensions_feegow_chrome\public e clicar em "Selecionar pasta"

Depois disso, a extensão já estará carregada e funcionando.

=================================================================================

DISPONIBILIZAR O ATUALIZADOR DA EXTENSÃO

1- Copiar o arquivo update-plugin.bat para a pasta de inicialização, que é possível ser aberta através de Win + r -> shell:start. Isso fará com que toda vez que a máquina for inicializada, o plugin será atualizado.

2 - Clicar com o botão direito no arquivo update-plugin.bat já dentro da pasta de inicialização e clicar em Enviar para -> Área de trabalho (criar atalho). Isso criará um atalho na área de trabalho e toda vez que for acionado, atualizará o plugin, mas sem a necessidade de reiniciar a máquina. O Chrome deve ser reiniciado.
